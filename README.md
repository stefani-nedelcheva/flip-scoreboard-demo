# Flip Scoreboard

A small vanilla JavaScript demo that mimics a split-flap style sports scoreboard.

## Features

- Pure CSS 3D flip animation
- Mock live score updates on an interval
- Stadium-inspired dark theme
- Static project ready for GitHub Pages

## Run locally

Open `index.html` in the browser, or serve the folder with any static server.

## Deploy

Push the repository to GitHub and enable GitHub Pages from the repository settings.
